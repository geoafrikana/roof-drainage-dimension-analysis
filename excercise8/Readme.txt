The folder conatins the folowing files:

- A microsoft word (.docx) file explaining the processes and results.

- Three raster (.tif) files:
    - idw_precipitation which contains the result of IDW interpolation.
    - NN_precipitation which contains the result of NN interpolation
    - Industrial_area_with_two_buildings_modified which is the georeferenced layer of the "screenshot taken with a camera".


- Excercise8.qgz file which the main QGIS project file.

- prec.csv which contains the weather stations and their locations in csv format.
